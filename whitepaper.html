Network request flow

A tun interface is created on each beendu node. The interface is assigned a IPv6 address based on the full beendu address of the node. The beendu address is the public key part of the node. We calculate sha1 hash and take least significant 112 bits and add fd00 to form the ipv6 address of the node.
ipv6_addr = "fd00"+ lsb(sha1(public_key), 112)

Packet router

The packet router is a usermode service that is attached to the tun interface. When a packet is arrived to the service it looks at the ipv6 packet header gets the destination address and route the packet to appropriate node.

DNS Server

The dns servers calculate the ipv6 address from the beendu address and return the dns response for any request.
